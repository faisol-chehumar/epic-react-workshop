การสร้าง React app ต้องการการตัดสินใจที่ expert ก่อนที่จะเขียน Code บรรทัดแรก หน้าที่ของคุณคือสร้างการสื่อสาร การดูแล codebase ที่จะช่วยให้ทีมสามารถสร้าง React app ที่ตรงกับความต้องการของ User ได้

คุณต้องทำการตัดสินใจเป็นร้อยๆอย่าง ถ้าคุณตัดสินใจได้ไม่ดี User จะต้อง suffer ทีมคุณจะต้อง suffer คุณอาจจะต้องติดอยู่การตัดสินใจแย่ๆ ไปอีก 5 ปี

React เป็นการตัดสินใจที่ ฉลาดสำหรับ web development ในปี 2020 แต่ความ flexibility ของมันก็อาจเป็นดาบสองคมได้เหมือนกัน มันขึ้นอยู่กับคุณและทีมของคุณที่จะต้องตัดสินใจว่าคุณจะสร้างและดูแล app อย่างไรเมื่อเวลาผ่านไป การตัดสินใจส่วนนี้ต้องการประสบการณ์

ปัญหาหนึ่งในการเรียนรู้ React คือเต็มไปด้วยบทเรียนที่ low-quality ตามเว็บ คุณจะต้องใช้เวลาหลายชั่วโมงเพื่อเรียนเนื้อหาที่ outdated ไม่ต่อเนื่อง เนื้อหาผิดๆ บทความเต็มไปด้วยคำแนะนำผิดๆ
